Outsight

Outsight is a macOS screen capture utility that lets you mirror secondary displays in a window on your primary display. Outsight features ultra-low latency, Picture-in-Picture, and multi-display support.

Download for macOS Requires macOS 13.0 or later
📺

Live Screen Capture

Monitor any display with ultra-low latency and minimal CPU usage.

🖼️

Picture-in-Picture

Keep an eye on your screens in a floating window while you work.

🧭

Multi-Display

Toggle between all connected displays seamlessly from the sidebar.
